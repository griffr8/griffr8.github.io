How to use

Set players "on ice"

Click a player when on ice. The box turns blue. Click again to deselect. You can do this while the timer is running.

The clock and periods

Click play icon to start timer. Click again to pause. Click the period end button to change period.

Save stats

When game is paused, you can save the statistics.
